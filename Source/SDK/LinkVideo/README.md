### 概述
Video SDK 主要提供了设备P2P接入、实时监控对讲、本地回放等常带电IPC基本能力。


## iOS 接入流程

1、在 App 的 Podfile 文件中添加如下依赖项

```
pod 'TIoTLinkKit/LinkVideo'
```
